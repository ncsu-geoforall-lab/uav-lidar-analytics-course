How to install Trimble Aerial Imaging software

  1. Download the installation file.
  2. The installer will guide you to install Trimble Installation Manager on your computer.
  3. Open the Trimble Installation Manager.
  4. Choose the Aerial Imaging (top left) and click Install (right bottom corner).
  5. In the next step you need to provide the username and password for the Trimble account, if you don't have one, you need to create it and then log in.
  6. Download should start automatically after logging in.
